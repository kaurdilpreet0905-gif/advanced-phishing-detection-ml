# 🚨 Advanced Phishing Detection System Using Machine Learning

## 📌 Project Overview
This project detects phishing websites using Machine Learning techniques.
A Random Forest classifier is trained on URL-based security features 
and deployed using Flask.

## 🧠 Features Used
- URL Length
- Dot count
- Presence of '@'
- HTTPS usage
- Hyphen count
- Subdomain count

## ⚙️ Technologies
- Python
- Scikit-learn
- Flask
- Pandas

## ▶️ How To Run

1. Install dependencies:
   pip install -r requirements.txt

2. Train model:
   python train_model.py

3. Run application:
   python app.py

4. Open browser:
   http://127.0.0.1:5000
